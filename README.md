# 🌤 Weather Forecast App

A modern and responsive **Weather Forecast Web Application** built with **React, Tailwind CSS, and Framer Motion**.  
The app allows users to search for any city and view the **current weather conditions and a 5-day forecast** with a clean animated UI.

---

## 🚀 Live Demo
https://weatherft.netlify.app/
---

# 📌 Features

✅ Search weather by city  
✅ Default city weather on page load  
✅ Current weather display  
✅ 5-day weather forecast  
✅ Weather icons and condition details  
✅ Humidity and wind speed information  
✅ Dark / Light mode toggle  
✅ Responsive design for mobile, tablet, and desktop  
✅ Animated UI using Framer Motion  
✅ Location suggestions using OpenWeather Geo API
